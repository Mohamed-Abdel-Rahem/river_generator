Flutter Joke App (Riverpod  + Code Generation)
A modern Flutter application demonstrating advanced state management using Riverpod  with Code Generation. This project fetches random jokes from a REST API and handles various application states gracefully.

Tech Stack & Tools
Flutter: Framework for building the UI.

Riverpod: For reactive state management.

Riverpod Generator: Used @riverpod annotations to generate type-safe providers.

HTTP: For networking and API integration.

JSON Serialization: To convert API responses into Dart models.

Key Features
Code Generation: Utilizing build_runner to generate providers automatically, reducing boilerplate and human error.

Async Lifecycle Management: Handling Loading, Data, and Error states using the .when() syntax.

Smart Refresh: Implementing "Pull-to-Refresh" and manual invalidation of providers using ref.invalidate() and ref.refresh().
